Upgrading

Recommended Docker Compose upgrade flow for Drakkar.

Recommended flow

docker compose pull
docker compose up -d --force-recreate
docker compose ps

When to use full stop/start

docker compose down
docker compose up -d

Use this when bind mounts changed, when troubleshooting stale networking, or when you want a full container restart path.

After upgrading

docker compose logs --tail 200 backend
docker compose ps
